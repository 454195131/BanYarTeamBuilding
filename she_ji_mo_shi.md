
#设计模式
####什么是设计模式？
一个模式是一个解决方案：一个在特定上下文里并且有着可预见的影响变化的问题的解决方案。（A pattern is a solution to a problem within a certain context that has a set of predictable consequences.）

描述这四个部分（问题、上下文、可预见的影响变化、解决方案），你就有了一个模式。
